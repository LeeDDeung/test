## 🛠️ 주요 Git 명령어 정리
| 명령어 | 설명 |
|--------|------|
| `git init` | 로컬 저장소 초기화 |
| `git add .` | 변경사항 스테이징 |
| `git commit -m "메시지"` | 커밋 생성 |
| `git push origin main` | GitHub로 푸시 |
| `git pull origin main` | 원격 저장소 변경사항 가져오기 |
